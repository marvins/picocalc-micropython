#  PicoCalc MicroPython Calculator

Written for PicoCalc using RP2350 Microcontroller

##  Features

-   Basic Calculator Functions

## Technical Details

- Language: MicroPython
- Microcontroller: RP2350

## Design

- `main.py` - Main Application
- `ui.py` - User Interface
    * `App_Runner` - Main Application Runner
- `apps/` - Application Directory
    - `main_menu.py` - Main Menu Application
    - `calculator.py` - Primary calculator application
- `utilities/` - Utility Functions
    - `gui_utilities.py` - Wrapper around PicoCalc Turtle API


## About Micropython

* No abc module
